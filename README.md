# E2E2-CATCHLINE
커널360 BE 2기 End to End 2팀


## 주제
- 맛집 원격 예약 서비스

## 기획 의도
- 해커톤보다 더 백엔드 아키텍처적으로 깊게 고민해보고 학습해보고자 함.
- 동시성 문제에 대해 경험해보고 해결해보고자 함.
- 비슷한 경험을 해볼 수 있는 서비스인 테이블링, 캐치테이블 앱을 참고하기로 함.

## 기능
1. 사용자
    - 손님, 사장님
2. 식당
3. 메뉴
4. 리뷰
5. 에약
6. 웨이팅
7. 스크랩

## 기술 스택
### 1. Back-end
    - JAVA 17
    - Spring Boot 3.3.2
    - Spring Data JPA
### 2. Front-end
    - Thymeleaf
    - HTML
    - CSS : bootstrap
### 3. Database
    - MySQL
